Github pages + Hexo 搭建个人主页

发表于 2016-12-02 | 分类于 Hexo | 阅读次数

　　尽量写一篇简洁的教程，照顾到很多想要有自己博客的同学。无论你从事什么专业，无论什么年龄，只要你想有一个属于自己的博客，都可以来看，会写的很详细，因为我和你一样，也是小白。
　　其实现在每个软件的官网都把安装步骤写的很详细了，只是即使这样，我们在安装的时候也会出现各种奇奇怪怪的错误，有时候Google也找不到和我们一样错误的人，你会不会觉得天啊，为什么只有我。所以我还是要写一下我安装的步骤，因为在这个过程中，我也遇到无数这样的坑，希望你们不要有坑，但如果有 ，我能帮到你们。
　　先解释一下这几个名词吧，解释的不好勿喷。

名词解释

Github pages

Github Pages 是依托 Github 本身的技术服务经验开发出来的一款产品，解决的是分布式的内容版本管理问题。架设博客是其功能的一部分，正如官网写的Websites for you and your projects.你可以在这里展示你的代码，也可做你想做的事情。

Hexo

Hexo是一个开源的静态博客生成器，用node.js开发，作者是台湾大学生tommy351。官网也有这样一句介绍：A fast, simple & powerful blog framework，因为我也是个小白，也不能说出hexo具体哪里好，但是简单方便是一定的，不然小白我也不能这么快就写个教程出来啊。

node.js

Hexo基于node.js，所以这是安装hexo不可少的环境。
Node.js 是一个基于 Chrome V8 引擎的 JavaScript 运行环境。这是官网的介绍，我觉得我不能更详细了，因为我对前端框架也不熟，在慢慢学习。

开篇

发表于 2016-12-01 | 分类于 日记 | 阅读次数

　　每次开篇都不知道怎么样开始，想说的很多，却难有个好开头。可能也想写写自己为什么会去折腾两天去做了自己的主页，这么说好像也不好，用陪我长大的昵称在godaddy申请了这个域名，然后和很多博主一样，用了github pages和hexo，并且用了next主题，自己做了一个搬运工，也有很强的愿望去自己租主机，自己写前端，只是暂时还不能花太多时间在这上面，之后会一点点做到的，下一篇也会像很多前辈一样，跟大家分享我在搭建这个博客一路上的坑，也希望能帮助之后的小伙伴吧。
　　
　　这一篇就讲讲故事吧。说起讲故事，还得给你们推荐一个正在更新的小说，黎明前的少年，这是一个很多年的朋友写的故事，他的故事，真实，质朴，就是发生在我们身上的那些故事，一个孩子成长必不可少的故事，最近他遇到麻烦，或许他会觉得一直麻烦不断，希望他能解决好一切吧。就像我们曾经经历过的，麻烦，也是故事。
　　
　　 我的故事开始吧。

　　本来也想写几个标题，类似小学到大学，也许是从小老师说流水账是很不好的形式，于是之后可能就会用成从鸡蛋到孵化成鸡这样的比喻，其实如出一辙。既然本来要写的就是流水账，那又不能用简单的标题，那干脆就不用了吧，想啥写啥，不小心成了意识流还能被人说好文笔。今天还是得有主题吧，得讲讲我为什么要做自己的主页，那就得讲讲，我是如何一定要走互联网这条路的。
　　
　　 分析任何事情，讲到原因，总离不开历史的推动，所以，我的小学生活开始了。总得说点和互联网相关的吧，那就是第一次进网吧了，那时候家庭条件不允许，并没有太接触过电脑，最多就是和姐姐去大妈妈的办公室，登个qq玩个连连看了，所以第一次和朋友一起去网吧，什么也不会玩，看着他们打着红警，我只能看看。很想玩啊，可是小时候的那个自闭少年并不愿意和人交流，只能默默看着自己的电脑屏幕。谁想到今天，无时不刻在和全世界的程序员交流，所以互联网改变了我。又跑了，接着在网吧。那个时候开始，就默默自己去学习电脑，那个年代所有网吧的电脑都是ghost一键装机的，于是顺理成章我也搞到一张dvd,应该是雨林木风的吧，不知道为什么当时的我就是不喜欢番茄花园，今天我也不喜欢吃番茄。后来自己家也终于有了电脑，开始折腾，亲朋家的电脑坏了，我就去重新装系统，没有ghost解决不了的事情。再到之后帮别人装机，等等，这个流水账好像真的无聊，我自己也写无聊了。也许想看到的剧本是天才计算机少年写出代码改变了世界。所以，这只是wlwlxgg的博客，我和你们一样，可能不是天才，但是有自己爱的东西，并且我们在努力，在分享。
　　
　　 初中就忽略吧，高中可以说一下，因为在高考最忙的时候，我居然在数学课上看谭浩强叔叔的c语言程序设计，其实当时并不知道为什么就想去学这个，只是百度做一个程序员需要学什么，然后就去图书馆借书了，就像今天，也依旧不知道自己为什么喜欢，可就是放不下。没过多久隔壁班的同学居然在看80x86汇编语言，大黑书，虽然那个时候不懂汇编是什么，但感觉好像也是计算机相关的，就也去捧了本回来，每天晚自习看着黑书，偶尔还做个课后习题百度一下答案，给自己当老师，很开心，只是第二天作业只能一顿抄。有本汇编语言是清华大学王爽编的，里面有一个题目类似于大家来找茬，说如果一眼看到问题所在的，你就有天赋，凑巧，我看到了，于是，心里又埋下一颗种子，我是有天赋的人，没想到到了大学学了电子专业，每天沉浸在电路板设计和硬件打交道的时候，汇编成了我的硬伤。说起大学生活很丰富，从一个正常人，胡吃海喝成了个胖子。认识了很多朋友，也做了很多事情。赚了一些小钱。春节和小伙伴去摆摊卖春联也成了美好的回忆。啊啊啊啊，转折点终于要来了。快乐而又腐败的大学生活的最后一年，我不知道为啥还是觉得，不能沉浸在怎么赚钱的世界里，我想写代码。于是，开始自学Android，PHP，从前端到后台，机缘巧合，碰巧跟朋友说我想写代码，做软件，他说他需要一款软件。（怎么表达起来这么无聊），然后我就答应他了，于是化身产品经理，或者说是一个中介的形象，去寻找团队，那个时候的自己还是天真烂漫的，一个需要超大并发的软件，自己还指望去找在校大学生做廉价劳动力自己能赚到多少报酬，可是事实证明，不可能。于是再去寻找更专业的人，付出更多的钱，始终没有理想的那样，有一个属于自己的团队。每天游走于甲方乙方之间，做着一个不知道在干嘛的事情，也许赚钱吧，可是我不能写代码。终于有一天自己承受不住这份压力，赔了自己的零用钱，其实到那个时候，我都不明白，我到底想要什么，我不知道为什么就有一个种子，时刻提醒我就该去写代码。朋友知道今天都在劝我说，你该创业的，你有能力有条件的。可是我还是拒绝了。
　　
　　 大学终于要毕业了，校园招聘也来了，上一阶段一直忙于交际应酬，没有写代码很久，变得生疏了，插一段今天同事还跟我讲的段子，一个国庆长假回来他就觉得手生了，已经有点不知道自己长假前写的东西了，这也是大家都会去写博客记录的原因吧，就像数学老师说一定要有错题集，虽然我没有过。再回来，各种笔试面试，都是知道但是记不起来了。于是结果很不理想。最终来了现在的公司做实习生。
　　
　　 终于讲完了，如果我是你，我肯定看不下去了，太枯燥无聊了，和那本黎明前的少年完全没有可比性。可是开篇，感觉还是得写多点文字，就这么瞎讲故事吧，都是故事啊。那么我为什么要去做这件事，哪件呢，文章开始说的，为什么要做自己的主页，前后呼应，还不错吧，只是没有过渡罢了。我那天跟妈妈说，我在互联网安了个家，我也有房了。现在很充实，很享受这样的生活，其实流水账过滤了很多精彩的内容，大多数就是经历了怎么样的苦，你要想聊，来哥们，我们喝酒吃肉聊可好。写文字总是写的写的就会脑子短路。就这么磕磕碰碰，我终于走上了一直想走的那条路，做了一个程序员，还是很弱的那种。好像还是没说出来，喜欢的原因，聊了这么久历史，可谁知道呢，就有一颗小种子，或许放弃了很多机会，成为现在社会大家所谓成功的人（有钱）。可是会后悔吗，种子还在，不管走了哪条路，终有一天它还要蹦出来把我拉回来。所以我不跟它反抗了，现在开始滋养它，慢慢让它发芽，这里就是那片土地，记录我，一个普通的人，和一颗不知道谁放进我心里的热爱code的种子之间的生活。
　　
　　 之后每天都会更新，工作中遇到的问题，听到的趣事，都会在这里和你们说，希望能和前辈帮助我一样帮助到你们，顺便说一下，今天用Markdown写了这个，有点激动。感觉又发现了好玩的东西。所以你看，我多么的小白，如果有一天，你觉得我的文章帮到了你，不用夸我，来看看这篇文章，原来我还不如你呢。
